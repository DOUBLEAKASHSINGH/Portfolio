# Akash Singh - Portfolio Website

A modern, responsive portfolio website built with Angular and Tailwind CSS, showcasing my journey as a Full Stack Developer, Data Analyst, and ML Enthusiast.

## Features

- Responsive Design: Optimized for desktop, tablet, and mobile devices
- Modern UI: Clean design with dark blue, black, and sky blue/beige color palette
- Smooth Animations: AOS (Animate On Scroll) library for engaging transitions
- Interactive Elements: Sticky navigation, mobile menu, and hover effects
- Contact Integration: EmailJS-powered contact form
- SEO Optimized: Meta tags and semantic HTML for better search visibility

## Tech Stack

- Frontend: Angular 18
- Styling: Tailwind CSS
- Animations: AOS (Animate On Scroll)
- Contact Form: EmailJS
- Deployment: Ready for Netlify/Vercel

## Sections

1. Navbar: Sticky navigation with smooth scrolling to sections
2. Hero: Introduction with name, tagline, and social links
3. About: Academic journey and motivation narrative
4. Skills: Technical skills grouped by categories with proficiency indicators
5. Projects: Showcase of key projects (DATA GENIE, NIRF Analysis, SocialApp, etc.)
6. Education: Academic timeline with institutions and achievements
7. Experience: Professional achievements and certifications
8. Contact: Contact form and social media links

## Quick Start

1. Clone the repository:
   ```
   git clone https://github.com/DOUBLEAKASHSINGH/Portfolio.git
   cd Portfolio
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the development server:
   ```
   ng serve
   ```


## Deployment

### Netlify
1. Connect your GitHub repository to Netlify
2. Set build command: ng build --configuration production
3. Set publish directory: dist/akash-portfolio
4. Deploy!

### Vercel
1. Connect your GitHub repository to Vercel
2. Vercel will auto-detect Angular settings
3. Deploy!
